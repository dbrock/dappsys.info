Dappsys

An open-source Solidity component framework

Create composable smart contract systems

Creating smart contracts for Ethereum today is harder than it needs to be. That’s why Nexus Development created Dapple and Dappsys. Dapple is a powerful developer multitool that allows you to write, test, and deploy your dapps to the blockchain all natively in Solidity. Dappsys is a standard library of building blocks that can be composed together to create many kinds of generic contract systems. Creating your system with Dappsys components makes it easier to focus on your dapp’s unique business logic and reason about complexity.

A product

ds-auth

Authentication

The need to properly secure each component in a multi-contract system running on the Ethereum platform is absolutely crucial. This package contains a range of primitives which can be combined to create any authentication scheme you need.
